Theme Toggle with SVG Demo

How it Works

This demo shows how to create a theme toggle that affects both your page styling and SVG diagrams. The key concepts are:

CSS Variables: Define theme colors in CSS custom properties that can be used by both HTML and SVG elements.

:root { --svg-primary: #007bff; --svg-stroke: #333333; } [data-theme="dark"] { --svg-primary: #0d6efd; --svg-stroke: #ffffff; }

SVG Classes: Apply CSS classes to SVG elements that use the CSS variables:

.svg-primary { fill: var(--svg-primary); } .svg-stroke { stroke: var(--svg-stroke); }

Theme Switching: Toggle a data attribute on the body/html element to switch between themes.
